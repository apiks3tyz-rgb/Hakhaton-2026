                    Адаптивний планувальник для студентів
Ця програма допомагає студентам балансувати між різними планами та завданнями..


Мова програмування: С++
База даних: локальний текстовий файл "tasks.txt"
Архітектура: Використана структура для зберігання даних у файл, алгоритм сортування,
збереження та зчитування даних через потік fstream

Пріоритет розраховується як відношення складності до терміновості з урахуванням ресурсів користувача.

Програма враховує стан (енергію - фізичну та ментальну), дедлайн(завдання, до яких менше 24годин,
отримують множник пріоритету x5), розподіл навантаження.. 

                            Щоб запустити: 
1) Скомпілюйте головний файл програми у Visual Studio
2) Якщо файл із завданням ще не створений, програма сама його створить
3) Обирайте дію з меню
